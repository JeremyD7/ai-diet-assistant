
# 添加 AI 分析功能 - 实施计划

## 需求分析
修改 `pages/index/index.vue`，添加对 Supabase `analyze-food` 函数的调用，支持通过图片或文字描述分析食物，并将结果传递到结果页面展示。

## 方案设计

### 1. 修改内容

**新增功能：**
- `startAnalyze` 方法：调用 Supabase Edge Function
- 修改现有的点击处理方法，让它们使用统一的分析流程
- 将分析结果传递给结果页面

### 2. 具体修改

**修改文件：** `pages/index/index.vue`

**修改步骤：**
1. 添加 `startAnalyze` 异步方法
2. 修改 `handlePreviewClick`：调用 `startAnalyze`
3. 修改 `submitDescription`：调用 `startAnalyze`
4. 分析成功后，将数据通过 `uni.navigateTo` 传递给结果页
5. 设置桶名为 'meal-images'（已在代码中）

## 核心功能代码

```javascript
const startAnalyze = async () => {
  if (!uploadedImageUrl.value && !foodDescription.value.trim()) return;

  uni.showLoading({ title: '分析中...' });
  try {
    const { data, error } = await supabase.functions.invoke('analyze-food', {
      body: {
        imageUrl: uploadedImageUrl.value || null,
        description: foodDescription.value.trim() || null,
      },
    });

    uni.hideLoading();

    if (error) {
      console.error('函数调用错误:', error);
      uni.showToast({ title: '分析失败: ' + error.message, icon: 'none' });
      return;
    }

    if (data.error) {
      uni.showToast({ title: 'AI错误: ' + data.error, icon: 'none' });
      return;
    }

    // 成功！传递数据到结果页面
    console.log('AI 分析结果:', data);
    
    // 跳转到结果页面，并传递数据
    uni.navigateTo({
      url: `/pages/result/result?data=${encodeURIComponent(JSON.stringify(data))}`
    });
  } catch (e) {
    uni.hideLoading();
    console.error('调用异常:', e);
    uni.showToast({ title: '请求失败', icon: 'none' });
  }
};
```

## 效果验证

1. 用户选择图片并上传后，点击预览图会调用 AI 分析

4. 分析成功后跳转到结果页面展示数据
