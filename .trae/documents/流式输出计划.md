
# 添加流式输出功能 - 实施计划

## 需求分析
为上传图片分析和文字输入分析添加流式输出功能，实现打字机效果，提升用户体验。

## 当前状态分析
- Edge Function 已能正常调用硅基流动 API
- 前端使用 `uni.showLoading` + `uni.showModal` 显示结果
- 非流式，用户需要等待完整结果才能看到

## 流式输出方案

### 方案选择：前端 Fetch + 流式处理
**推荐理由**：
- `supabase.functions.invoke` 不直接支持流式
- 直接 fetch Edge Function URL 可获得完整的流式能力
- 更灵活，可自定义流式处理

### 技术实现

#### 1. Edge Function 修改
```typescript
// 添加流式参数支持
const { imageUrl, description, stream } = await req.json();

// 在调用硅基流动 API 时启用流式
body: JSON.stringify({
  model: "Qwen/Qwen2.5-7B-Instruct",
  messages: [...],
  stream: true,  // 启用流式
})

// 获取流式响应
const reader = response.body?.getReader();
const decoder = new TextDecoder();

// 流式传递给客户端
const stream = new ReadableStream({
  async start(controller) {
    while (true) {
      const { done, value } = await reader.read();
      if (done) break;
      controller.enqueue(value);
    }
    controller.close();
  }
});
```

#### 2. 前端修改
```typescript
// 使用 fetch 直接调用 Edge Function
const response = await fetch('https://xxx.supabase.co/functions/v1/analyze-food', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${accessToken}`,
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({ imageUrl, description, stream: true }),
});

// 处理流式响应
const reader = response.body.getReader();
const decoder = new TextDecoder();

// 实时更新 UI
let fullContent = '';
while (true) {
  const { done, value } = await reader.read();
  if (done) break;
  
  const chunk = decoder.decode(value);
  // 解析 SSE 格式数据
  const lines = chunk.split('\n');
  for (const line of lines) {
    if (line.startsWith('data: ')) {
      const data = JSON.parse(line.slice(6));
      fullContent += data.content || '';
      // 更新 UI 显示
      updateDisplay(fullContent);
    }
  }
}
```
