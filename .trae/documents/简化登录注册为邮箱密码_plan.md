# 登录注册简化为邮箱+密码 - 实施计划

## 📋 需求分析
简化登录注册流程，统一使用邮箱+密码，移除手机号字段，降低用户注册门槛。

## 📁 项目现状
- 文件：`pages/login/login.vue`
- 登录表单：邮箱或手机号（第58行）+ 密码（第74行）
- 注册表单：邮箱（第140行）+ 手机号（第152行）+ 密码 + 确认密码 + 协议同意

## 🎯 目标状态
- 登录表单：仅邮箱 + 密码
- 注册表单：邮箱 + 密码 + 确认密码 + 协议同意
- 移除所有手机号相关字段和验证

## 🔧 实施步骤

### 步骤 1：修改登录表单
**文件**：`pages/login/login.vue`

#### 1.1 更新 placeholder（第58行）
- **当前**：`placeholder="请输入邮箱或手机号"`
- **修改为**：`placeholder="请输入邮箱"`

#### 1.2 更新提示文本（第257行）
- **当前**：`title: '请填写完整信息'`
- **保持不变**：这个提示已经很合适

### 步骤 2：修改注册表单
**文件**：`pages/login/login.vue`

#### 2.1 删除手机号输入框（第145-155行）
删除整个手机号输入框：
```vue
<view class="input-group">
	<view class="input-wrapper">
		<text class="input-icon">📱</text>
		<input 
			class="input-field" 
			type="number" 
			placeholder="请输入手机号" 
			v-model="registerForm.phone"
		/>
	</view>
</view>
```

#### 2.2 删除 data 中的 phone 字段（第225行）
从 `registerForm` 中移除 `phone: ''`

#### 2.3 删除手机号验证逻辑（第277-280行）
删除以下代码：
```javascript
if (!this.registerForm.phone) {
    uni.showToast({ title: '请输入手机号', icon: 'none' });
    return;
}
```

### 步骤 3：优化布局
注册表单字段减少后，可以适当调整：
- 减小表单卡片内边距（从56rpx调整为48rpx）
- 保持所有输入框间距一致性
