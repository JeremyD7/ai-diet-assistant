
# 改进错误处理和调试 - 实施计划

## 问题分析
当前遇到错误："AI 返回格式不正确，无法提取 JSON"，接口返回500错误。需要改进错误处理以更好地调试。

## 改进方案

### 1. 增强错误处理和调试信息

**修改内容：**
- 添加详细的请求和响应日志
- 打印完整的 data 对象内容
- 更好地解析和显示错误信息
- 添加请求参数的日志
- 改进错误提示，让用户知道具体问题

### 2. 关键修改点

```javascript
const startAnalyze = async () => {
  if (!uploadedImageUrl.value && !foodDescription.value.trim()) return

  // 打印请求参数
  const requestParams = {
    imageUrl: uploadedImageUrl.value || null,
    description: foodDescription.value.trim() || null,
  }
  console.log('分析请求参数:', requestParams)

  uni.showLoading({ title: '分析中...' })
  try {
    const { data, error } = await supabase.functions.invoke('analyze-food', {
      body: requestParams,
    })

    uni.hideLoading()

    // 打印完整的响应数据
    console.log('完整响应 - data:', data)
    console.log('完整响应 - error:', error)

    if (error) {
      console.error('函数调用错误:', error)
      // 显示更详细的错误信息
      uni.showToast({ 
        title: error.message || '分析失败', 
        icon: 'none',
        duration: 3000
      })
      return
    }

    if (data?.error) {
      console.error('AI返回错误:', data.error)
      uni.showToast({ 
        title: 'AI错误: ' + data.error, 
        icon: 'none',
        duration: 3000
      })
      return
    }

    // 检查必要字段是否存在
    if (!data?.total_calories) {
      console.warn('返回数据格式异常:', data)
      uni.showToast({ 
        title: '返回数据格式异常', 
        icon: 'none',
        duration: 3000
      })
      return
    }

    // 成功！显示结果
    console.log('AI 分析结果:', data)
    uni.showModal({
      title: '分析结果',
      content: `热量: ${data.total_calories} kcal\n蛋白质: ${data.protein_g}g\n脂肪: ${data.fat_g}g\n碳水: ${data.carbs_g}g\n评分: ${data.health_score}/10\n建议: ${data.advice}`,
      showCancel: false,
    })
  } catch (e) {
    uni.hideLoading()
    console.error('调用异常 - 完整错误:', e)
    console.error('错误消息:', e.message)
    uni.showToast({ 
      title: e.message || '请求失败', 
      icon: 'none',
      duration: 3000
    })
  }
}
```

### 3. 调试步骤

1. 在浏览器开发者工具的控制台中查看日志
2. 检查请求参数是否正确
3. 检查完整的响应数据
4. 根据错误信息判断是前端还是后端问题

## 预期效果

- 更详细的错误日志便于调试
- 更好的用户体验，了解具体错误
- 能快速定位问题所在
